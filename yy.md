# 北漂故事01
刚来不久认识的一个妹子，偶尔一起看电影逛公园，学工科的，浑身散发着一股严谨务实之风，人挺漂亮的但不善打扮。一个周末，她肚子不舒服一天没怎么吃东西，我正好在公司处理个bug，离她住处很近，问她要不要吃点东西，我会做点家常饭菜，她说没啥胃口但同意这个提议。我就去超市买些果蔬过去，做了三菜一汤，她吃得倒是开心。一直阴沉的天空，还是在傍晚下起了雨。
：雨挺大，别回了吧，你睡沙发可以吗？她抱着一个毛毯。  
：你不怕吗？  
：有一点，但你刚才那句“去躺着吧，碗筷我来洗”有被暖到。  
：晚安。  
雨听到一半，她跑过来钻进怀里。  

# 毕业旅行1
独自去的西北，青旅十来个人在楼顶睡帐篷，大家天南海北胡侃。注意到有个自驾的妹子，刚工作不久的小学老师，夏天嘛，穿的很清凉，她皮肤特别白，微胖丰满，简直极品。得知她教语文，我便搜遍脑子里为数不多的古文历史地理知识，给大家喂点儿文青们感兴趣的话题，啤酒、吉他、故事，夜与星光。
第二天很早我就出发了，打算在这里再逛一天就去下一站。约莫俩小时后，突然在街上听到有人在喊我的外号，四处张望看到是那个老师，在身后不远处探出车窗摆着手喊我，她示意我上前。  
：去哪呀弟弟？  
：随便逛逛，今天没有目的地。  
：走，上车，敦煌！  
：好！  
大西北，一条蜿蜒的公路蔓延向前消失在天地辽阔处，车辆不多，都疾驰而过，合着澎湃的音乐，心情也变得摇晃起来。行至一处破旧房屋，我们停下来煮了些吃的，然后她提议在此小憩片刻，起身去拿凉席毯子过来。自然是要敦伟大友谊的，兴至高出，彻底坦诚相见。相拥小憩后，她提议，要不就这样继续赶路吧。于是我们俩就这样大摇大摆地收拾东西启程。一路上兴致来了就敦，就这样一路到敦煌，一口气吃了十几个杨柳大串儿，6瓶啤酒下肚，回去又疯狂一阵子，美美地睡了十多个小时。醒来的时候她已经走了，我们前一天就说好不必告别。
背起包，我又往下一个城市赶去。  

# 屋檐下01
公司隔壁组一个妹子，业务上有些交集，偶尔一起吃饭。有一天同事凑局一起去酒吧，她喝得有点多，点名让我送她回去。好容易给她送到家扔在床上盖好被子，我就上个洗手间的功夫，回来看她已经扒拉干净四仰八叉躺着，别说身材还挺好，驻足欣赏了一会儿她的╰(☫)╯，小心给她盖好毯子。然后躺在客厅沙发上睡去。朦胧间天还没亮，恰好晨起时刻她坐了上来。

# 屋檐下07
有过业务往来的同事妹子找到我，说有个idea需要一个开发帮她写代码，开了个不错的价格，约咖啡馆聊下细节，聊完我觉得没多少工作量，大概俩周末就能搞定。她惊呼以为要挺久，说就是合作过觉得挺靠谱才找我。这是背景。  
前几天的晚上我们就找个会议室头脑风暴一下，聊聊进展和想要进一步调整的地方，效率非常高。周末傍晚她问我在干嘛，我说在会议室写了10个小时代码(实际上一半时间在看剧)。她觉得很不好意思，要请我吃饭，说要亲自下厨，我接受了这个提议，于是去健身房冲个凉，路上拐到超市挑一瓶葡萄酒带上。  
她的厨艺不错，我们找了部电影边看边吃。吃完饭电影才放到1/3，便继续看，此时突然两人也没话说，略显尴尬，我就起身说去收拾下桌子碗筷，她没阻止，说去洗个澡，头发上有油烟味，让我收拾好先别走再聊些细节，并拿出些水果让我待会儿边吃边看电影。  
看着电影，她喊我过去，问我能不能帮她吹头发。她没换睡衣，而是裹着浴巾，其实到这里我大概知道她什么想法，就顺她的意吧。过程中她拿梳子梳理着，裹着的浴巾就慢慢松开，我假装不没看到。果然就滑落了，我假装慌张，她淡定表示不碍事，继续。嗯，身材很不错，很圆，很白，而且除头发其余地方毛发应该都已剔除干净。当然，这种场合我需要保持淡定，也就没有任何过分的举动，继续拿着吹风机给她吹头发。完事儿她蹲下捡浴巾的时刻，伸手摸了摸我的𓂸，露出邪魅一笑，没等我有所动作就把我裤子扒了。。。嗯，👄很厉害。  
折腾到大半夜，她说不习惯旁边有人，得，又是睡沙发。清晨又是一场激战。  
第二天下午，我还是在会议室，想把一个todo搞定。她问我在干嘛，然后就非要过来。一开始我们还在画板聊东西，后面就干脆在会议室里疯狂了一把。体验确实刺激。
