# 合租往事2
刚毕业合租，一间次卧是一个男生，也是互联网大厂，几乎没见过面，我住里面的次卧，主卧是一对情侣。客厅不大，平常基本都窝在各自的房间，几无交流。主卧情侣深入交流的时候会开比较大声音的音乐，但感觉每次都没多久就结束。  
正值盛夏，每天要冲个澡才好睡觉，为避免尴尬，我每次都是穿好短裤短袖去洗漱。主卧哥们比较随意，很多时候只穿个内裤，可能他觉得无所谓吧，毕竟这里唯一的异性是他女朋友。偶尔会和小姐姐在狭窄的洗手间门口有一点为数不多的交集，没有沟通。她的睡衣都很清凉半透，开口也都比较低。偶尔交错的一刻离得较近，仗着身高优势，俯视能稍微看到她挺拔圆润的两个半球白嫩如雪，加上身上香味很好闻，真是让人浮想联翩气血翻涌。后来我开始只穿内裤和显身材的短袖，长期锻炼跑步的身材在普通人里也算是不错的黑皮体育生级别。往后的几次相遇，我能感受到她开始偷瞄我，心中暗喜。  
那几天正构思着以一个什么样的理由尝试搭讪，她竟然主动通过合租小群加我，俩人似是而非正经地聊几句，再无话。又过两天傍晚，她朋友圈更新一条：男朋友出差的第二天...虽然不确定是不是暗示，管他呢，直接发了一张比较阳刚的照片过去，正忐忑中，她回了句门没锁。壮壮胆子，径直推门进去，她穿的是纯黑色的情趣内衣！彼时作为一个刚毕业的热血男生哪受得了这场面，一股热流直冲脑门的冲击感。她很直接，褪去我的内裤半跪着开始口，几秒钟就坚挺的要爆炸，看得出她很欣喜，主动趴在阳台上撩起睡裙，没想到那里光滑的很，应该是刚剃的，此时已经很湿润，水汪汪已经流出来到大腿上，一杆进洞，她啊的一声，叫的很大声很欢喜。那晚的音乐喧嚣了很久。

# 合租往事5
夏初跳槽来到杭州，好山好水好风光，人也比北方水润许多。  
刚来的两周住酒店，把在这边的一个同学和来之前在网上聊得火热的俩妹子陆续喊来深入交流。找房子初期很想住在附近村子里的农房，瓜田月下，很有生活气息，怎奈潮湿蚊虫难忍，只好作罢。最后还是选择合租，三个房间，主卧一男一女，朝南次卧有人但出差较多，我住朝北次卧。  
住处唯一的妹子打扮极其新潮靓丽狂野，高高瘦瘦，古铜色的皮肤应该是专门晒出来的，很好看。人很热情，刚加到合租小群，她就打招呼，晚上还单独过来问要不要去吃夜宵，我很纳闷她怎么不和“男友”一起，就随口问一下，她倒坦诚，说那不是男友，只是她的搭子。我对这类关系见怪不怪，但此时意识到是个机会，于是就说好啊，随即一同出门。撸串+啤酒，她酒量不错，简单一顿饭，了解到一些基本情况，美院美术生在读，刚分手不久，搭子也是艺术生无业在此租房，其他信息听完就忘了。最后付款我没让她A钱，她说下次她请。  
日子平淡而飞逝，接下来的一两周我们没有什么交集，除了一次她在线问我会不会被他们晚上运动声音打搅，我说有降噪耳机。直到周末，我本来在公司加班，妹子语音问能不能回去看下她的猫咪有没有在客厅，她有点担心。猫咪？我怎么没听到过叫声，但还是答应她回去看一眼，就说大概15分钟到。打开门直接一个震惊，他俩在客厅沙发云雨中，一丝不挂，啪啪水声混着嘤咛呻吟！尽管也在野外 学校之类的地方搞过，但直面这种场景还是有点困惑：他们什么意思，我该怎么办？  
看我在门口踟蹰，男生眼神示意我往桌子上看，一盒TT。懂了，于是平复一下情绪往客厅走去，但内心还是挣扎，毕竟没有过多人经验，很忐忑不安。走到跟前我也不知道该怎么办，脱裤子吗？旁边有个男的，感觉非常尴尬。手也无处安放一般，僵硬地揉了揉她的胸，不大，但很有弹性。无法直视他们任何一个人，尴尬情绪压过欲望，我打算回房间或者出门，此时妹子却一把抓住我的胳膊使劲儿掐，呻吟声也更大了，一声销魂的“啊~~”直接给我血脉翻涌上头~  
艹！直接把邦儿硬的𓂸塞到她嘴里，俩手使劲儿揉搓双乳，男生此时也兴奋冲刺，给妹子干得哦啊不止。随后男生示意换位置，我撕开一个TT戴上，噗呲插进去，紧、润、柔软而温暖，她拉起长音，“噢喔————”。接着换姿势，妹子弯腰吃男生，我后入她，一顿疯狂输出，给她干跪在地。接着我前胸贴妹子后背抱起她，扒开双腿面对男生，他活塞几分钟缴械。我继续把妹子转过身来，抱着正入，一阵缓插速攻，给妹子怼得呻吟带着哭腔喷了一地。此时把妹子放在沙发上，撸下TT，𓂸塞进她嘴里，汩汩射进去，单手摁头把妹子脸贴在我的三角区不让她拔出来，另一只手两根手指插入嫩穴疯狂搅弄，直到咕噜咕噜吞咽下去才放她瘫软在沙发上。
