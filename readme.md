# 合租往事2
刚毕业合租，一间次卧是一个男生，也是互联网大厂，几乎没见过面，我住里面的次卧，主卧是一对情侣。客厅不大，平常基本都窝在各自的房间，几无交流。主卧情侣深入交流的时候会开比较大声音的音乐，但感觉每次都没多久就结束。
正值盛夏，每天要冲个澡才好睡觉，为避免尴尬，我每次都是穿好短裤短袖去洗漱。主卧哥们比较随意，很多时候只穿个内裤，可能他觉得无所谓吧，毕竟这里唯一的异性是他女朋友。偶尔会和小姐姐在狭窄的洗手间门口有一点为数不多的交集，没有沟通。她的睡衣都很清凉半透，开口也都比较低。偶尔交错的一刻离得较近，仗着身高优势，俯视能稍微看到她挺拔圆润的两个半球白嫩如雪，加上身上香味很好闻，真是让人浮想联翩气血翻涌。后来我开始只穿内裤和显身材的短袖，长期锻炼跑步的身材在普通人里也算是不错的黑皮体育生级别。往后的几次相遇，我能感受到她开始偷瞄我，心中暗喜。那几天正构思着以一个什么样的理由尝试搭讪，她竟然主动通过合租小群加我，俩人似是而非正经地聊几句，再无话。又过两天傍晚，她朋友圈更新一条：男朋友出差的第二天...虽然不确定是不是暗示，管他呢，直接发了一张比较阳刚的照片过去，正忐忑中，她回了句门没锁。壮壮胆子，径直推门进去，她穿的是纯黑色的情趣内衣！彼时作为一个刚毕业的热血男生哪受得了这场面，一股热流直冲脑门的冲击感。她很直接，褪去我的内裤半跪着开始口，几秒钟就坚挺的要爆炸，看得出她很欣喜，主动趴在阳台上撩起睡裙，没想到那里光滑的很，应该是刚剃的，此时已经很湿润，水汪汪已经流出来到大腿上，一杆进洞，她啊的一声，叫的很大声很欢喜。那晚的音乐喧嚣了很久。
